\doxysubsection{Detailed Description}
A flexible wrapper class for neural networks that inherits from torch\+::nn\+::\+Module. 

This class provides a convenient interface for creating, training, and using neural networks with Py\+Torch C++. It encapsulates common neural network operations and provides easy-\/to-\/use methods for training and inference.
