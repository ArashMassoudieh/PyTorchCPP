Detailed Description

template<class T>
class GeneticAlgorithm< T >

Template class implementing a Genetic Algorithm.

This class provides a general-purpose implementation of a genetic algorithm for model optimization. The type parameter T typically represents the model type being optimized.

Template Parameters
TType of the model or object being optimized.
