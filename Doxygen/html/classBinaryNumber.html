Detailed Description

Represents a binary number and provides utilities for conversion, mutation, crossover, and manipulation.

A BinaryNumber stores its value as a string of '0' and '1' characters, and provides helper functions to convert between decimal and binary, perform crossover operations, mutate bits, and split into segments.
